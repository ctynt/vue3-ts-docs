# 创建应用

## 1. 使⽤ Vite 创建 Vue 3 项⽬

```bash
npm create vite@latest
```

在提示中，按照以下步骤选择：

- 输⼊项⽬名称，例如 vue-basic
- 选择框架为 Vue
- 选择语⾔为 TypeScript

或者直接使⽤ Vite 的 Vue-TypeScript 模版命令⼀键安装：

```bash
npm create vite@latest vue-basic -- --template vue-ts
```

## 2.  进⼊项⽬⽬录并安装依赖

```bash
cd vue-basic
npm install
```

## 3.  启动开发服务器

```bash
npm run dev
```
